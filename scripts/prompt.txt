あなたの仕事は、日本語学習用の短い例文を1件だけ生成することです。

必ず JSON オブジェクトを1つだけ返してください。
JSON以外の文字を一切出力してはいけません。
前置き、後書き、説明、Markdown、コードブロックは禁止です。

出力形式:
- JSON object
- キーは必ず text, study_point, translation_en の3つだけ
- 3つの値はすべて文字列
- キーの追加、削除、改名は禁止
- 各値の中に改行を入れてはいけません

今回の生成用シード:
{{PROMPT_SEED}}

今回使う文型コード:
{{PATTERN_CODE}}

文型コードの対応:
- A: 理由（〜ので / 〜ため）
- B: 義務・必要（〜なければならない / 〜ねばならない）
- C: 禁止・不許可（〜てはいけない / 〜ことは許されない）
- D: 推量・様態（〜らしい / 〜ようだ）
- E: 逆接（〜のに / 〜にもかかわらず）
- F: 不可・制約（〜わけにはいかない）

重要:
- 必ず今回の文型コードに対応する文型を1つだけ使ってください
- 複数の文型を混ぜすぎないでください
- study_point では、その文型を必ず説明してください
- シードは出力に書かないでください
- 「毎回違う文を書こう」と考える必要はありません。今回の文型コードに従って自然な1文を作ってください

text の条件:
- 自然な日本語の例文1文だけを書く
- 必ず「うんこ」を含める
- 12文字以上にする
- 「例文:」「学習ポイント:」「英訳:」などのラベルを書かない
- 箇条書きにしない
- 説明文やメタ発話を書かない
- 文末記号は1個だけにする
- 文末は必ず「。」「！」「？」のいずれかにする
- 半角ピリオド "." や "．" で終えない
- 不自然な文、意味不明な文、壊れた文を書かない

study_point の条件:
- 必ず1文だけで書く
- 必ず「『」と「』」を含める
- 「『...』」の中には、text に実際に含まれている表現を書く
- ただし N2文型として「〜」を用いた形で示してもよい
- 今回の文型コードに対応する表現を説明する
- その表現の意味または使い方を、日本語で簡潔に説明する
- 「学習ポイント:」などのラベルを書かない
- 単語1語だけで終わらせない
- 文末記号は1個だけにする
- 悪い例は避ける:
  - 単語だけ
  - 『』がない
  - text にない表現を説明する
  - ただの語句メモ

translation_en の条件:
- text 全体の自然な英訳を1文で書く
- 「Translation:」「英訳:」「English:」などのラベルを書かない
- 改行しない
- 学習ポイントの説明を混ぜない

出力は JSON オブジェクトのみです。
